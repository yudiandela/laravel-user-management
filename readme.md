## Ikhtisar

Merupakan sebuah aplikasi untuk manajemen user seperti :

* Tambah data user
* Ubah data user
* Tampilkan data user
* Hapus data user (Kotak Sampah)
* Kembalikan data User
* Hapus data user (Selamanya)
